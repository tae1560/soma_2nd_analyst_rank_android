Fixing this part :

MyWebview’ is missing required library: ‘C:\Users\name\Desktop\GoogleAdMobAdsSdk-6.4.1.jar’

 

Right Click on the project >> Build path >> configure build path >>Libraries >> Add external Jars >> point GoogleAdMobAdsSdk-6.4.1.jar

remove  the x mark .jar file .. you are done

You can find the jar file inside the downloaded folder

or you can download it from here

Export

 

Right click on the project folder > export > export > export android application

default debug.keystore location is My Documents > .android > debug.keystore

default password is "android" ( without quotes )

If you have created your own keystore file then remember, DONOT FORGET THE debug.keystore PASSWORD.

Links

 

Youtube Link

To run youtube videos you must give hyperlinks like this
 
vnd.youtube://videoid
 
 

Text hyperlink

example :<a href="vnd.youtube://DoDOdwvLikQ> Youtube Sample Link </a>

 

Image hyperlink

<a href="vnd.youtube://videoid"><img src="http://yourimage.com/1.jpg"></a>


Market Link

 
To direct your user to rate your app or to download some 
other apps from android market give hyperlink like this
 market://details?id=com.my.webview
 

Dial Link - Touch To Call

To make call directly from your app use this hyperlink
 
tel:123456789
example : <a href="tel:123456"> call us </a>

Use this for twitter and facebook linking

If you use https:// it will be redirect to browser..

 

One touch Mail

Use the following code for one touch mail . it will open default gmail app with your email address in TO field

<a href="mailto:mail@yoursite.com">contact us</a>

Play mp3,Mp4,3gp files

All the hyperlinks which ends with mp3 will play in a popup ( tested in ICS 4.0.4 )

*Must be direct links

<a href="http://yoursite.com/preview.mp3">contact us</a>

<a href="http://yoursite.com/preview.mp4">contact us</a>

<a href="http://yoursite.com/preview.3gp">contact us</a>

 Location :

To locate your workplace to your app users, you must use latitude and longitude of that place

geo:latitude,longitude

 

Example : Location : <a href="geo:40.688969, -73.881798"> Locate us </a>

All webview app developers use only http://
